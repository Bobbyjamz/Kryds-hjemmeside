Systemflows — ende-til-ende

✓ OK
  1. ✓Besøgende ser forsiden tracked i analytics
  2. ✓Besøgende klikker branche → quick-book modal → sender forespørgsel (e-mail til kontakt@krydsbyg.com)
  3. ✓Besøgende tilmelder sig som medarbejder på /tilmeld (foto + CV som base64 i e-mail)
  4. ✓Admin logger ind på /admin og ser besøg + medarbejdere + vagter + feed
  5. ✓Admin bekræfter medarbejder → 6-cifret kode sendes via Resend
  6. ✓Medarbejder logger ind på /medarbejder/login med telefon + kode
  7. ✓Medarbejder ser feed + åbne vagter + tager en vagt
  8. ✓Admin opretter feed-besked → broadcast-e-mail til alle medarbejdere
